# COVID-19 Power BI Dashboard

Interactive dashboard built with Power BI to analyze global COVID-19 trends.

## Features

- Global KPIs
- World map visualization
- Country rankings
- Time-series analysis
- Continent filtering

## Tools

- Power BI
- DAX
- Power Query

## Dataset

Source: Our World in Data (OWID)
